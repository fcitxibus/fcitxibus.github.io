6月22日随笔

发表于 2021-06-22 更新于 2021-06-23 分类于 生活随笔

　　　　今天去外出了，在到达客运站的时候竟然忘记带身份证了，还好车站有临时身份证打印机器，解决了我的燃眉之急。

　　　　今天和客户谈话的时候，竟然没有那样的紧张慌张的感觉，没有之前的练习，也就没有自己今天的坦然，实践出真章；

　　　　今天有以下几个思考：

　　　　1. 我的工作是不是无意义的，我可以从我工作中学习到什么；

　　　　2. 我该怎么去学习，向哪些人去学习，如果我不去思考，那我的工作将毫无意义；

　　　　3. 接下来我需要把我的工作做到什么程度（了解，掌握，熟练，精通）；

　　　　4. 在做一件事的时候，该怎么把这件事与其一些准备工作，细节相连，串通起来(联动，思维不设限)；
　　　　5. 什么是复利？

复利 ：做事情A，会导致结果B；而结果B，又会反过来加强A，不断循环。

  1. 简单的事情重复做，重复的事情认真做，如此而已。
  2. 很多人认为复利的核心在于回报率，但其实复利真正的核心在于时间。
  3. 新知识不断成为下一次思考素材的积累，从而让知识能够不断以“复利”速度快速迭代。
  4. 新知识不断成为下一次思考素材的积累，从而让知识能够不断以“复利”速度快速迭代。
  5. 人脉复利，损害朋友利益，就是从银行取钱，帮助朋友，回以微笑，就是存钱，雪球会越滚越大。
  6. 真正的智者，追求稳健增长，更看重长期性的时间，而不是短时间的回报率。
  7. 很多事情，都是坚持到后面有了临界点，才会爆发出效果，复利思维作为人生的第一策略，要求你具有的第一品质就是：耐心，坚持。只要方向是对的，不要焦虑和不安，多给自己一些时间。
  8. 复利思维作为人生策略的另一个要求是：不要透支，无论健康、人脉、财富、智力，都不要去透支。
  9. 目标感愈强烈，复利思维对你人生的影响会越大。
  10. 别盲目相信勤奋的力量，熬夜加班和你躺在床上睡大觉，很多时候没有多大区别。
  11. 你始终要记得，复利的人生，更看重有生活质量的人生，因为只有这样的人生，才能走得长远。
  12. 你始终要记得，复利的人生，更看重有生活质量的人生，因为只有这样的人生，才能走得长远。

总结：如果总是让自己待在自己的舒适圈，拒绝学习新知识，去交流，做无用的思考，道路会约来越窄，只有活水涌进来，新水和保持池塘的流动性才能越发感觉充实。

多了解别人的观点（尤其是你抗拒的，不喜欢的那些）。

端午节后第一天

发表于 2021-06-16 分类于 生活随笔

​ 今天是端午节后第一天上班，上午开了周例会，主要有几件事记忆深刻：

  1. 项目专员进行总结的时候条理清晰，目标明确，能很好地表述自己上周的工作内容，以及本周的工作安排；
  2. 经理说把写重复性劳动，还有工作流程，做出细致到每一步的工作流程（包括QA检查）；
  3. 自己暗生骄傲，洋洋自得，殊不知是夜郎自大，坐井观天，拘泥于一处。
  4. 工作不够认真，没有流程，规章制度。
  5. 工作中该进行总结，找出最利于自己的方式，同时要不断进行复盘。
  6. 我该感谢我的朋友，我的父母（永远不要忘记拥有一颗感恩之心）！

记一次端午节搬家

发表于 2021-06-15 分类于 生活随笔

端午节

      今天是2021年6月14日，农历五月初五，中国的传统节日端午节。

      因为工作地点和租房太远，每天要经历3个小时的往返路程，我终于决定在端午假期期间—搬家，搬到离公司稍近一点的地方去住，就在前两天我去找了找房子，下午去，天略微有点热，端午放假第一天，跟从自己的内心，去了xxxx小区，开始还以为是一个几栋矮一些的老旧小区，随之大汗淋漓的赶到之后，是A、B、C、D四栋，旁边还有学生公寓，看了下公寓环境，网上搜索不给退押金，遂怕，就行离开。

      随后骑着单车不知不觉看到一个老小区，想着宅胖胖上一次的租房，鬼使神差的走进小区门，为了下那大爷，果然他是在等我，随后现任房东过来带我看房，初始看房很满意，感觉就是自己理想的房子，红漆实木老式家具，那一瞬就吸引了我的眼球，心里告诉自己：“租下他”，但是房东毫无绝情的泼了我冷水—房租很贵，我犹豫之后走了，但是心里却未放下对它的念想。

      昨天晚上根据宅胖的情报，在那的小区找了找，也是整租，价格优惠了很多，但是心底还是有着落差，XX小区的阿姨很好，但是房子没能留住我。然后又看了一家，这一家感觉房东（户主）有一些问题，遂离开了。之后给现任房东打电话，定下来了，红色的实木家具，我想像的样子，虽然知道它并不好。

      端午来了，我搬家了，一番来回折腾一段晴，一段雨的路程，我来到了，现在这个房子。

      我终究还是来了，不知道对错，不知道适宜，我租下了它，给了自己一个暂时的居所。你好！
